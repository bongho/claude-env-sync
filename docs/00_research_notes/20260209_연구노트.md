## 주요 결정 사항

1. **Python + Click 선택**: Bash 대신 Python으로 구현. JSON 처리, LLM 통합 가능성, 크로스 플랫폼 지원이 이유.
2. **GitPython 사용**: subprocess 대신 GitPython 라이브러리 사용으로 안정적인 Git 연산.
3. **Tier 기반 동기화**: 파일을 3개 Tier로 분류하여 MVP에서는 Tier 1+2만 동기화.
4. **시크릿 사전 검사**: push 전 API 키 패턴 탐지하여 자동 차단.
